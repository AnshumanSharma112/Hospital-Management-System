 Overview :
The Hospital Management System is a web-based application developed using Java Spring Boot that helps manage hospital operations efficiently.
It provides functionalities for managing patients, doctors, appointments, billing, and administrative tasks in a centralized system.

Features
👨‍⚕️ Doctor Management
Add, update, delete doctor details
View doctor availability & specialization
🧑‍🤝‍🧑 Patient Management
Register new patients
Update patient records
Maintain patient history
📅 Appointment Scheduling
Book appointments with doctors
Manage schedules
Avoid conflicts
💊 Prescription & Treatment
Record diagnosis and treatment
Store prescriptions digitally
💳 Billing System
Generate bills
Track payments
🔐 Authentication & Authorization
Secure login system
Role-based access (Admin, Doctor, Patient)
